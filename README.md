# 🍽️ Food Donation System (Java OOP)

The **Food Donation System** is a Java-based application built using **Object-Oriented Programming (OOP)** principles. It aims to reduce food waste by connecting donors with people or organizations in need.

---

## 🚀 Features

* Add and manage food donations
* Request available food items
* Role-based system (Donor / Receiver / Admin)
* Track donation status (Pending, Approved, Delivered)
* Clean and structured OOP design

---

## 🧠 OOP Concepts Used

* **Encapsulation**
* **Inheritance** 
* **Polymorphism** 
* **Abstraction**
---

## 🛠️ Technologies Used

* Java
* OOP Principles
* (Optional: File Handling / Database)

## 📦 Installation

1. Clone the repository:

```bash
git clone https://github.com/your-username/food-donation-system.git
```

2. Navigate to the project folder:

```bash
cd food-donation-system
```

3. Open the project using your IDE:

* IntelliJ IDEA / Eclipse / NetBeans

4. Compile and run the project:

```bash
javac Main.java
java Main
```

---

## ▶️ Usage

1. Run the application

2. Choose your role:

   * Donor → Add food donations
   * Receiver → Request food
   * Admin → Manage donations

3. Follow the on-screen menu to interact with the system
